# MathTaskClassLibrary

## Цель работы
Изучить создание и использование модульных тестов для библиотеки классов.

## Описание
Проект включает:
- Основной проект `MathTaskClassLibrary`, который реализует метод для вычисления площади прямоугольника, объёма цилиндра и площади многоугольника.
- Проект с тестами `MathTaskClassLibraryTests`, который проверяет правильность работы методов.

## Как запустить тесты
1. Откройте решение в Visual Studio.
2. Перейдите в окно "Обозреватель тестов".
3. Выполните тесты.
